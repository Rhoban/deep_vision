# README #

### What is this repository for? ###

* Experimenting vision with deep learning

### How do I get set up? ###

* Requires tiny-dnn https://github.com/tiny-dnn/tiny-dnn (submodule) and OpenCV
* git submodule init
* git submodule update
* mkdir build
* cd build
* cmake ..
* make
